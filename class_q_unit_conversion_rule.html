Detailed Description

template<typename String>
class QUnitConversionRule< String >

The QUnitConversionRule class represents a unit conversion rule from base unit to second unit within the specified family. Note that in QUnitConvertor class each family can be represented with the single base unit. In other words you can't have two unit conversion rules with the same family but different base units.
